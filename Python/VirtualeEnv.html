Virtual environment python:

Come prima cosa si deve avere python installato

Come creare un ambiente virtuale:

  1. una volta scaricato python si apre il prompt dei comandi come amministratori poi si esegue il comando "pip install virtualenv"
  2. poi si esegue il commando "pip installvirtualenv"
  3. Ora che abbiamo tutto instllato possiamo creare l'ambiente tramite il comando "mkvirtualenv NAME"
  4. Adesso è tutto pronto per eliminare l'ambiente si esegue il comando "rmvirtualenv NAME"
  5. Per attivare l'ambiente virtuale si esegue il comando "rmvirtualenv NAME" però l'ambiente deve esistere perchè altrimenti darebbe questo messaggio
